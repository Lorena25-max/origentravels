# Proyecto Origen Travels

# Plataforma de Tours en Medellín

Aplicación web desarrollada con React que permite explorar, consultar y gestionar información de tours en Medellín y sus alrededores.  
El proyecto está diseñado con una arquitectura escalable y componentes reutilizables, siguiendo buenas prácticas de desarrollo frontend profesional.


## Objetivo del Proyecto

Construir la base estructural del frontend con una arquitectura organizada que facilite:

- Escalabilidad del sistema
- Trabajo colaborativo en equipo
- Integración futura con backend
- Navegación clara para el usuario
- Buenas prácticas de SEO

Este avance corresponde al primer momento del proyecto integrador.



## Tecnologías Utilizadas

- React (componentes funcionales)
- Vite
- JavaScript + SWS
- CSS3
- React Router DOM
- Git y GitHub



## Estructura del Proyecto
src/
├── assets/ # Recursos estáticos (imágenes, estilos globales)
├── components/ # Componentes reutilizables de UI
│ ├── layout/ # Estructura base (Navbar, Footer, Layout)
│ └── tours/ # Componentes específicos del dominio
├── helpers/ # Funciones utilitarias
├── pages/ # Vistas principales del usuario
├── router/ # Configuración de rutas
├── services/ # Comunicación con backend (preparado para uso futuro)
├── App.css 
├── App.jsx # Orquestador principal
├── index.css # Orquestador principal
└── main.jsx # Punto de entrada de React

## Buenas Prácticas Aplicadas

- Componentes en PascalCase
- Componentes funcionales únicamente
- Estructura escalable
- SEO básico con estructura semántica
- Código limpio y modular
- Preparación para consumo de API
- Organización profesional del repositorio


## Instalación y Ejecución

### Requisitos
- react-router-dom
- Node.js 18 o superior
- npm 
- Git

### Pasos

bash
# Clonar repositorio
git clone <URL_DEL_REPOSITORIO>

# Entrar al proyecto
cd nombre-del-proyecto

# Instalar dependencias
npm install
npm react-router-dom



# Ejecutar servidor de desarrollo
npm run dev
